# Security Policy

| Version | Supported          |

| 0.05    | :white_check_mark: |

| Thing | Meaning |

## Reporting a Vulnerability

### If you can fix it:
Please fix the issue and create a pull request.
### If you can't fix it:
Please create an issue.
